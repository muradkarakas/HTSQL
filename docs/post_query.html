"post_query" trigger

Just before sending content of the data block row to the client, post_query trigger is fired.
post_query trigger function is optional. if it is needed, must be in Code Behind File.
post_query trigger has 1 variants.
post_query trigger is a function with a specific name. Trigger function and its name should conform to the following format:
Function prototype:

void PostQueryTriggerFunctionName();

Function's name format:

blockName + ".post_query"

What can be done in post_query trigger function:

  • Setting formula column's value.

Example:

void deps.post_query() {
}
