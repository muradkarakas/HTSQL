Installation:

  1. For Visual Studio:
    • Backup original files.
    • Copy and paste 2 files into that folder:
      "C:\Program Files (x86)\Microsoft Visual Studio 14.0\Common7\Packages\schemas\html"
      The path above is subject to change depending on the version of Visual Studio you have.
    • Go to :
      Tools \ Options \ Text Editors \ File Extensions
    • Add a new entry and use following values
      Extension: Editing Experience:
      frmx HTML (Web Forms) Editor with Encoding
    • Restart Visual Studio
