populate_tree

populate_tree function has 1 variants:

void populate_tree(char treeName);

populate_tree function initializes the tree element. After successful initializing, "tree_node_expanded" Trigger will be fired.

treeName property is a string property and should have match the following format:
block name + "." + tree id
