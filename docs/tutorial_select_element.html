Datalist & Select Element Usage

This demo page shows:

  • how to use Data List tag, populate_datalist function and record set variables
  • how to fill a drop down list with a Data List tag
  • how to define a look up select element

Create a new sub folder named selectElement under the site\pages folder of the installation path.

Form File:
Create a file named selectElement.frmx and copy/paste the code block showed below into the file.

1 <!DOCTYPE html>
2 
3 <html>
4 
5  <head>
6  <title>Select Element Demo</title>
7  </head>
8 
9  <body>
10 
11  <datalist id="provinces"></datalist>
12 
13  <datalist id="counties"></datalist>
14 
15  <controlblock control-block-name="cb1">
16  <select style="width: 100px" name="province_id"
17  datalist-name="provinces">
18  </select>
19  </controlblock>
20 
21  <controlblock control-block-name="cb2">
22  <select style="width: 100px" name="county_id" datalist-name="counties"
23  lookup-input-block-name="cb1" lookup-input-name="province_id">
24  </select>
25  </controlblock>
26 
27  </body>
28 </html>

Code behind File:
Create a file named selectElement.sqlx and copy/paste the code block showed below into the file.

1 
2 void page.load() {
3 
4  rsProvinces = select province_name as label, province_id as value
5  from
6  provinces
7  order by
8  province_name;
9  populate_datalist('provinces', rsProvinces);
10 
11  rsCounties = select county_name as label, county_id as value, province_id as province_id
12  from
13  counties
14  order by
15  county_name;
16  populate_datalist('counties', rsCounties);
17 }
18
