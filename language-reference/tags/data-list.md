# Data List

## Description

* Data lists are used to populate select tags.
* Cannot be nested.
* [populate\_datalist](../built-in-functions/sodium-built-in-functions/other-functions/populate_datalist.md) function is used to fill data list.

## Declaration

```text
<datalist
    id                          = ""
    connection-name             = "">
</datalist>
```
